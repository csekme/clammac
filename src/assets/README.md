# App-on belüli logó

Tedd ide `logo.svg` (ajánlott) vagy `logo.png` néven — a sidebar brand-blokkja
a következő buildnél / dev-újraindításnál automatikusan ezt jeleníti meg a
pajzs-jelvény helyett. Amíg nincs itt fájl, a zöld/sárga pajzs marad.

- 36×36 px-en jelenik meg, 10px lekerekítéssel — négyzetes arányú kép a jó.
- SVG esetén skálázási gond nincs; PNG-nél legalább 72×72 (Retina).

A tray-, Dock- és csomagolt app-ikonok NEM ide valók — azok helyét a
`resources/icons/README.md` írja le.
